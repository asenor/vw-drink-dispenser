# VW Drink Dispenser

## Installation

1. Install Java 11 [SDKMAN](https://sdkman.io/)

2. (Optional) Install just:
    ``` bash
    brew install just
    ```

## How to run all tests

Using just:
``` bash
just test
```

Using gradle build:
``` bash
./gradlew build
```
